# 找单独的数

# 问题描述

在一个班级中，每位同学都拿到了一张卡片，上面有一个整数。有趣的是，除了一个数字之外，所有的数字都恰好出现了两次。现在需要你帮助班长小C快速找到那个拿了独特数字卡片的同学手上的数字是什么。

要求：

设计一个算法，使其时间复杂度为 O(n)，其中 n 是班级的人数。 尽量减少额外空间的使用，以体现你的算法优化能力。

# 测试样例

  • 样例1：
输入：cards = [1, 1, 2, 2, 3, 3, 4, 5, 5]
输出：4
解释：拿到数字 4 的同学是唯一一个没有配对的。
  • 样例2：
输入：cards = [0, 1, 0, 1, 2]
输出：2
解释：数字 2 只出现一次，是独特的卡片。
  • 样例3：
输入：cards = [7, 3, 3, 7, 10]
输出：10
解释：10 是班级中唯一一个不重复的数字卡片。

# 约束条件

  • 1 ≤ cards.length ≤ 1001
  • 0 ≤ cards[i] ≤ 1000
  • 班级人数为奇数
  • 除了一个数字卡片只出现一次外，其余每个数字卡片都恰好出现两次

# 解题思路

由于题目要求时间复杂度为 O(n)，且需要尽量减少额外空间的使用，我们可以利用位运算中的异或操作来解决这个问题。

异或操作具有以下性质：

  • 任何数和 0 做异或运算，结果仍然是原来的数，即 a ^ 0 = a。
  • 任何数和其自身做异或运算，结果是 0，即 a ^ a = 0。
  • 异或运算满足交换律和结合律，即 a ^ b ^ a = (a ^ a) ^ b = 0 ^ b = b。

根据以上性质，我们可以将数组中的所有数字进行异或运算，最终得到的结果就是那个只出现一次的数字。

# 代码实现

# python
def find_unique(cards):
    unique = 0
    for num in cards:
        unique ^= num
    return unique

// go
func findUnique(cards []int) int {
    unique := 0
    for _, num := range cards {
        unique ^= num
    }
    return unique
}

# 复杂度分析

  • 时间复杂度：O(n)，其中 n 是数组 cards 的长度。我们需要遍历整个数组一次，进行 n 次异或运算。
  • 空间复杂度：O(1)，我们只使用了常数级别的额外空间来存储结果。

# 测试样例

cards = [1, 1, 2, 2, 3, 3, 4, 5, 5]
print(find_unique(cards))  # 输出：4

cards = [0, 1, 0, 1, 2]
print(find_unique(cards))  # 输出：2

cards = [7, 3, 3, 7, 10]
print(find_unique(cards))  # 输出：10
