# Ubuntu Server Install Nvidia Driver

# 添加Graphics Drivers PPA

PPA（Personal Package Archive）是一种用于在Ubuntu中安装软件的方式。输入以下命令将Graphics Drivers PPA添加到系统中：

$ sudo add-apt-repository ppa:graphics-drivers/ppa

按下回车后，系统可能会要求你输入管理员密码。

# 更新软件包信息

执行以下命令来更新软件包信息：

$ sudo apt update

# 安装NVIDIA驱动

安装NVIDIA显卡驱动的命令会根据你的显卡型号和系统需求而有所不同。一般来说，你可以使用以下命令来安装推荐的稳定版本驱动：

$ sudo apt install nvidia-driver-<版本号>

在上面的命令中，将<版本号>替换为你想要安装的驱动版本号。例如，如果要安装版本号为470的驱动，命令将是：

$ sudo apt install nvidia-driver-470

# 重启系统

安装完成后，重新启动系统以使新的驱动生效。
