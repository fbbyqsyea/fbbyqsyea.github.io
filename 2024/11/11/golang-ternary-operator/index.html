# 三元运算符

Go语言中并没有内置的三元运算符（类似a ? b : c），但可以通过定义一个泛型函数 Ter 来实现相同的效果。以下是详细的实现与使用示例。

# 实现三元运算符函数

在Go 1.18+版本中引入了泛型，允许定义泛型函数。Ter 函数的实现如下：

// 三元运算符函数
func Ter[T any](cond bool, a, b T) T {
	if cond {
		return a
	}
	return b
}

# 参数解析

  • 泛型类型参数 T：[T any] 表示函数可以接受任意类型的参数 T，这里的 any 是 Go 中的一个关键字，表示任何类型。
  • 函数参数 cond：cond 是一个 bool 类型的参数，表示条件是否满足。
  • 参数 a 和 b：a 和 b 是类型为 T 的两个值，即当 cond 为 true 时返回 a，否则返回 b。

# 返回值

函数返回值为 T 类型，与 a 和 b 的类型一致。

# 使用示例

Ter 函数的用法与其他语言的三元运算符非常相似。以下是一些使用示例：

package main

import (
    "fmt"
)

func Ter[T any](cond bool, a, b T) T {
	if cond {
		return a
	}
	return b
}

func main() {
    // 示例1：数值类型
    x := 10
    y := 20
    max := Ter(x > y, x, y)
    fmt.Println("最大值:", max)  // 输出: 最大值: 20

    // 示例2：字符串类型
    greeting := Ter(true, "Hello", "Goodbye")
    fmt.Println("问候语:", greeting)  // 输出: 问候语: Hello

    // 示例3：自定义类型
    type User struct {
        Name string
        Age  int
    }

    user1 := User{Name: "Alice", Age: 25}
    user2 := User{Name: "Bob", Age: 30}
    selectedUser := Ter(user1.Age > user2.Age, user1, user2)
    fmt.Printf("选择的用户: %+v\n", selectedUser)  // 输出: 选择的用户: {Name:Bob Age:30}
}
